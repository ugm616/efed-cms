üîê Authentication

Most endpoints require authentication. The system uses:

  • Session-based authentication with secure HTTP-only cookies
  • CSRF protection for all state-changing operations
  • Role-based access control with 5 user levels:
    • Viewer (read-only access)
    • Contributor (can create content)
    • Editor (can edit content)
    • Admin (can delete content and manage users)
    • Owner (full system access)
  • Optional Two-Factor Authentication with TOTP

üìù Data Schema

The system manages the following entities:

  • Wrestlers: Individual performers with records, ratings, and statistics
  • Companies: Wrestling organizations and promotions
  • Divisions: Weight classes, gender divisions, and competition categories
  • Events: Shows, pay-per-views, and wrestling events
  • Matches: Individual contests between wrestlers
  • Tags: Flexible categorization system for all entities
